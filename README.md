# python-employee-app

Employee Tracking App written in Python &amp; Flask for BF&amp;C

## Requirements

- Python 3.11.3

## Installing dependencies

```shell
cd web
pip3 install flask
pip3 install pandas
```

## Running the Application

### Terminal Client and Server

```shell
python3 run.py
```

### Web-based Interface

```shell
cd web
python3 web.py
```
